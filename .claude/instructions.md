# Інструкції для Claude Code

## Загальні правила роботи з проєктом

### Мова та кодування

1. **Робота з українською мовою:**
   - Весь текст українською мовою ОБОВ'ЯЗКОВО має бути у кодуванні UTF-8
   - Перед створенням або редагуванням файлів з українським текстом завжди перевіряй правильність кодування
   - Після створення файлу з українським текстом перевір, чи він коректно відображається

2. **Перевірка кодування:**
   - Використовуй команду `file` для перевірки кодування існуючих файлів
   - Якщо файл не в UTF-8, видали його та створи заново з правильним кодуванням
   - При будь-якому україномовному запиті користувача перевіряй цей файл інструкцій

3. **Створення файлів:**
   - Завжди використовуй UTF-8 кодування для всіх текстових файлів
   - Для файлів .md (Markdown) особливо важливо зберігати UTF-8
   - Після створення файлу з українським текстом запропонуй користувачу перевірити відображення

## Структура проєкту

Це проєкт курсової роботи з психології, що містить дослідження про:
- Риси особистості (модель Великої П'ятірки)
- Арт терапію
- Зв'язок між рисами особистості та сприйняттям арт терапії

## Типові завдання

1. **Створення документації:**
   - Всі документи мають бути українською мовою
   - Формат: Markdown (.md)
   - Структура: заголовки, підзаголовки, списки, таблиці
   - Обов'язкові секції: Вступ, Основний зміст, Висновки, Джерела

2. **Пошук наукової інформації:**
   - Використовуй WebSearch для пошуку наукових досліджень
   - Шукай peer-reviewed статті
   - Вказуй посилання на джерела (DOI, PMC, назви журналів)
   - Надавай року публікації

3. **Робота з даними:**
   - Структуруй інформацію у зрозумілому форматі
   - Використовуй таблиці для порівняння
   - Виділяй ключові висновки
   - Вказуй рівень доказовості досліджень

## Чеклист перед завершенням завдання

- [ ] Перевірити кодування файлу (має бути UTF-8)
- [ ] Перевірити, що весь текст українською мовою
- [ ] Переконатися, що структура документа логічна
- [ ] Додати джерела та посилання на дослідження
- [ ] Запропонувати користувачу перевірити відображення файлу

## Приклад перевірки кодування

```bash
# Перевірити кодування файлу
file "назва_файлу.md"

# Правильний результат:
# назва_файлу.md: UTF-8 Unicode text

# Неправильний результат (потрібно перестворити):
# назва_файлу.md: data
```

## Важливо

⚠️ **При кожному україномовному запиті:**
1. Прочитай цей файл інструкцій
2. Слідуй усім вказаним правилам
3. Особливо звертай увагу на кодування UTF-8
4. Після створення файлу перевір його кодування

## Контакт

Якщо виникають проблеми з кодуванням або відображенням тексту:
1. Видали проблемний файл
2. Створи його заново з правильним UTF-8 кодуванням
3. Перевір командою `file`
4. Підтверди з користувачем, що текст відображається коректно
